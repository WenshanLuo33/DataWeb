Exploring Urban Data through Web Mapping

This dataset presents the percentage of female legislators in parliaments worldwide, based on data from Our World in Data. Analyzing this data helps to understand the progress and disparities among countries in promoting gender equality and women’s political participation. By applying different classification methods—such as quantiles, equal intervals, natural breaks, and standard deviation—we can visualize which countries excel in gender representation and which still face significant gaps. Additionally, this analysis provides valuable insights for policymakers to foster more inclusive political environments. To enhance spatial representation, I will apply three different projection maps, allowing for a comparative perspective on how data is interpreted across varying cartographic views.

All data processing and analysis were conducted in my Colab notebook .
